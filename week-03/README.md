## LISTS

**Add Numbers →** Create an empty list called `numbers`. Add the values 10, 5, 20, and 15 using `append`. Display the list.

**Remove Value →** Given the list `[10, 20, 30, 40, 50]`, remove the value 30 using `remove`. Display the result.

**Sort Numbers →** Create a list `[7, 3, 9, 1, 5]`. Sort the list in ascending order, then sort it in descending order. Display both results.

**Manage Names →** Create an empty list called `names`. Add 5 names using `append`. Remove one specific name using `remove`. Sort the list using `sort` and display the final result.

**List Operations →** Given the list `[8, 3, 8, 1, 9, 3]`, remove the first occurrence of the number 3. Add the number 10 using `append`. Sort the list and display the final result.
