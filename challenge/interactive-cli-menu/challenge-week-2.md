### Challenge: Interactive Terminal Menu

You must develop a program in a programming language of your choice that displays an interactive menu in the terminal, allowing the user to select different options until they decide to exit the program.

#### 📋 Requirements:

1. When the program starts, it should display a menu with at least 4 options, for example:
   - 1: Display a welcome message
   - 2: Calculate the sum of two numbers
   - 3: Check if a number is even or odd
   - 4: Exit

2. The program must keep running using a loop (e.g., `while`), showing the menu repeatedly after each action until the user chooses to exit.

3. For each option:
   - **Option 1:** Display a simple message on the screen.
   - **Option 2:** Ask the user for two numbers and display their sum.
   - **Option 3:** Ask for a number and inform whether it is even or odd.
   - **Option 4:** End the program with a goodbye message.

4. The program must handle invalid input:
   - If the user enters an invalid option, display an error message and show the menu again.
   - Ensure numeric inputs are valid when required.
