### IF/ELSE

### 1. Even or Odd

Ask the user to enter an integer.

- Display whether the number is **even** or **odd**.

### 2. Age Check

Ask the user to enter their age.

- If the age is **18 or older**, display: "Adult"
- Otherwise, display: "Minor"

### 3. Compare Two Numbers

Ask the user to enter two numbers.

- Display which one is **greater**
- Or inform if they are **equal**

### 4. Student Grade

Ask the user to enter a grade (from 0 to 10).

- If the grade is **7 or higher**, display: "Approved"
- Otherwise, display: "Failed"

### 5. Password Check

Ask the user to enter a password.

- If the password is **"1234"**, display: "Access granted"
- Otherwise, display: "Access denied"

### FOR

1. **Print numbers from 1 up to a given number**
   → The program should ask the user for a number and use a `for` loop to print all numbers from 1 to that value.

2. **Read 5 names and display them**
   → The program should ask the user to input 5 names and then print each one using a `for` loop.

3. **Read 5 numbers and count the even ones**
   → The program should ask for 5 numbers and use a loop with an `if` statement to count how many are even.

4. **Display the multiplication table of a number**
   → The program should ask for a number and print its multiplication table from 1 to 10 using a `for` loop.

5. **Read 5 numbers and find the largest**
   → The program should collect 5 numbers and determine which one is the largest using `if` conditions.
