### 1. Even or Odd

Ask the user to enter an integer.

- Display whether the number is **even** or **odd**.

---

### 2. Age Check

Ask the user to enter their age.

- If the age is **18 or older**, display: "Adult"
- Otherwise, display: "Minor"

---

### 3. Compare Two Numbers

Ask the user to enter two numbers.

- Display which one is **greater**
- Or inform if they are **equal**

---

### 4. Student Grade

Ask the user to enter a grade (from 0 to 10).

- If the grade is **7 or higher**, display: "Approved"
- Otherwise, display: "Failed"

---

### 5. Password Check

Ask the user to enter a password.

- If the password is **"1234"**, display: "Access granted"
- Otherwise, display: "Access denied"
