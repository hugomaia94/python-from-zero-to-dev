### IF/ELSE

1. Even or Odd
   → Ask the user to enter an integer. Display whether the number is even or odd.

2. Age Check
   → Ask the user to enter their age. If the age is 18 or older, display "Adult", otherwise "Minor".

3. Compare Two Numbers
   → Ask the user to enter two numbers. Display which one is greater or if they are equal.

4. Student Grade
   → Ask the user to enter a grade (0 to 10). If it is 7 or higher, display "Approved", otherwise "Failed".

5. Password Check
   → Ask the user to enter a password. If it is "1234", display "Access granted", otherwise "Access denied".

### FOR

1. Print numbers from 1 up to a given number
   → Ask the user for a number and use a for loop to print numbers from 1 to that value.

2. Read 5 names and display them
   → Ask the user to input 5 names and print each one using a for loop.

3. Read 5 numbers and count the even ones
   → Ask for 5 numbers and count how many are even using a loop and if statement.

4. Display the multiplication table of a number
   → Ask for a number and print its multiplication table from 1 to 10 using a for loop.

5. Read 5 numbers and find the largest
   → Ask for 5 numbers and determine the largest using if conditions.

### WHILE

1. Simple counting
   → Use a `while` loop to print numbers from 1 to 10.

2. Sum of numbers
   → Ask the user to enter numbers until they type 0. Display the sum of all entered numbers.

3. Multiplication table
   → Ask the user for a number and display its multiplication table from 1 to 10 using a `while` loop.

4. Countdown
   → Ask the user for a number and display a countdown from that number to 0.

5. Password validation
   → Ask the user to enter a password until it is correct ("python123"). When correct, display "Access granted".
